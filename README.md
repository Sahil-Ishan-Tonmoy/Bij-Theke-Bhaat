# Bij Theke Bhaat (বীজ থেকে ভাত) 🌾
[![Flutter](https://img.shields.io/badge/Flutter-v3.11+-02569B?logo=flutter&logoColor=white)](https://flutter.dev)
[![Firebase](https://img.shields.io/badge/Firebase-v9.0+-FFCA28?logo=firebase&logoColor=black)](https://firebase.google.com)
[![Gemini AI](https://img.shields.io/badge/Gemini%20AI-2.5%20Flash-4285F4?logo=google&logoColor=white)](https://ai.google.dev)

**Bij Theke Bhaat** (From Seed to Rice) is a cutting-edge, AI-powered agricultural ecosystem designed specifically for rice farmers in Bangladesh. It bridges the gap between traditional farming wisdom and modern generative AI to maximize crop yield, reduce waste, and simplify financial management.

---

## ✨ Key Features

### 🔍 AI Disease Scanner
- **Real-time Detection**: Upload or take a photo of rice leaves to identify diseases like Brown Spot, Blast, or Bacterial Blight instantly.
* **Expert Treatment**: Receives personalized, professional treatment plans generated by **Gemini 2.5 Flash**.
- **Confidence Scoring**: Transparent AI certainty metrics for every diagnosis.

### 🌱 Smart Soil Health Hub
- **Unified Nutrient Analysis**: Log pH, Nitrogen (ppm), and Moisture levels to receive a Scoped status (Critical, Good, or Optimal).
- **Generative Strategy**: Get a 60-word "Expert Agronomic Strategy" customized for your specific soil profile.
- **Visual Trends**: Interactive graphs to track soil health changes over time.

### 💧 Weather-Aware Irrigation Scheduler
- **Precision Watering**: Calculates exact water volume (L/Bigha) based on crop growth phase and day count.
- **Safety Overrides**: Automatically detects rain or storms via the **Open-Meteo API** and suggests skipping irrigation to prevent root rot.

### 🏪 Glassmorphic Marketplace
- **Direct-to-Buyer**: List your crop yield with pricing, quantity, and location.
- **One-Tap Dialing**: Seamless "Call Seller" integration with auto-sanitized phone numbers for instant communication.
- **Premium UI**: Sleek, modern design with real-time Firestore updates.

### 📊 Financial & Yield Management
- **Profit/Loss Calculator**: Automated calculation based on harvest weight and market rates.
- **Expense Tracker**: Track fertilizer, labor, and seed costs with categorized visualizations.
- **Bilingual Interface**: Fully supports **English** and **Bengali (বাংলা)** for maximum accessibility.

---

## 🛠️ Technology Stack

- **Frontend**: Flutter (Dart) with a modern Glassmorphic UI design system.
- **Backend**: Firebase Firestore (NoSQL) & Firebase Auth (Secure Sign-In).
- **AI Engine**: Google Gemini 2.5 Flash / 2.0 Flash (Multi-model fallback architecture).
- **Real-time Data**: Open-Meteo Weather API & Firebase Cloud Messaging.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (v3.11 or higher)
- Android Studio / VS Code
- A Google Gemini API Key

### Installation
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Sahil-Ishan-Tonmoy/Bij-Theke-Bhaat.git
   ```
2. **Install Dependencies**:
   ```bash
   flutter pub get
   ```
3. **Configure Firebase**:
   - Add your `google-services.json` to `android/app/`.
4. **Run the App**:
   ```bash
   flutter run
   ```

## 🛡️ Architecture Highlights
- **High-Availability AI**: Implemented a sequential fallback system (Gemini 2.5 Flash -> 2.0 Flash -> 2.5 Flash-Lite) to handle API quota limits gracefully.
- **Smart Caching**: Local `SharedPreferences` caching for AI responses to reduce redundant API hits and save data.
- **Adaptive UI**: Theme-aware components that automatically adjust to user settings and light/dark preferences.
